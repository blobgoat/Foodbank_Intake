How to use Figma documentation:

You should arrive at this screen:

![Figma icon top left, then tab icon, then comment icon. Below the top left corner of icons there is a list of flow states, start should be selected. after the left third of the screen there is a zoomed out portion of the window (like pdf viewer) Which shows a phone interface of the design for foodbank manager](figma_design_doc_images/enter_figma.png)

You can interact with the website as if it were your phone to explore all the graphic design I am proposing. If at any point you have a comment suggestion, or something which needs to be changed. Feel free to use the comment feature to add comments (the speech bubble on the top left).

you then click where you want to comment:
![Clicking the comment icon third to the top left. A speech bubble. And clicking the interface will result in a comment text editor to pop up. The image shows this flashing text](figma_design_doc_images/add_comment.png)

and add your information or tidbit.
![The user is typing in a complement to the designer in the text editor and about to push the up arrow which represents post](figma_design_doc_images/writing_comment.png)

Leaving the comment accessible for the design team to incorporate feedback.
![The user posted their comment and a blue speach icon with their account icon is added to the interface. In addition there is a previously empty space which now has one comment to demonstrate it has now been appended to a list of comments easily accesible to everyone](figma_design_doc_images/acessible_comment_created.png)

I went ahead and fleshed out almost everything as it seems like volunteers time is limited. But don't let that dissuade you from giving criticism.
